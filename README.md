# PROFINAL
첫번째 안드로이드 작품입니다.

## 목차
* 사용기술
* 주요기능

## 사용기술
* **OS** : Android

* **언어** : Java

* **데이터베이스** : SharedPreferences

* **라이브러리** : gson
